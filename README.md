# Insurance-Premium-And-Payout-KPI-Dashboard
This project focuses on end-to-end data analysis and interactive dashboard creation using Power BI. The objective of the project is to transform raw data into meaningful insights that help stakeholders understand trends, performance metrics, and key business indicators for better decision-making.
